## Välkommen till Sajn API

Sajn API ger dig programmatisk tillgång till alla funktioner i Sajn-plattformen. Med vårt API kan du skapa, hantera och signera dokument, samt administrera kontakter, företag och anpassade fält.

## Autentisering

Alla API-anrop kräver autentisering via API-nyckel som skickas i `Authorization`-headern:

```bash
curl -H "Authorization: your-api-key" https://app.sajn.se/api/v1/documents
```

## Bas-URL

```
https://app.sajn.se/api/v1
```

## Huvudfunktioner

### Dokument
Skapa, hämta, uppdatera och ta bort dokument. Hantera signeringsprocessen och ladda ner signerade dokument.

### Kontakter
Hantera din kontaktlista för att enkelt lägga till signerare till dokument.

### Företag
Skapa och hantera företagsinformation.

### Taggar
Organisera dina dokument med anpassade taggar.

### Anpassade fält
Skapa och hantera anpassade fält för dina dokument.
