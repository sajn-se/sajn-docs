---
title: "Activity Events"
description: "All document activity event types returned by the API"
---

Every action on a document is recorded as an activity event. You can retrieve these events via `GET /api/v1/documents/:id/activity`. Each event has a `type` field with one of the values listed below.

## Delivery Events

| Event | Description |
|---|---|
| `EMAIL_SENT` | Signing invitation email was sent |
| `SMS_SENT` | Signing invitation SMS was sent |
| `REMINDER_SENT` | Signing reminder was sent |
| `DELIVERY_BOUNCED` | Email delivery bounced |
| `DELIVERY_FAILED` | Email delivery failed |
| `DELIVERY_DELIVERED` | Email was delivered |
| `DELIVERY_COMPLAINED` | Email was marked as spam |

## Signer Events

| Event | Description |
|---|---|
| `SIGNER_CREATED` | A signer was added to the document |
| `SIGNER_UPDATED` | A signer's details were updated |
| `SIGNER_DELETED` | A signer was removed from the document |

## Document Lifecycle Events

| Event | Description |
|---|---|
| `DOCUMENT_CREATED` | Document was created |
| `DOCUMENT_SENT` | Document was sent for signing |
| `DOCUMENT_OPENED` | Document was opened by a signer |
| `DOCUMENT_READ` | Document was read by a signer |
| `DOCUMENT_VIEWED` | Document was viewed |
| `DOCUMENT_SIGNED` | Document was signed by a signer |
| `DOCUMENT_REJECTED` | Document was rejected by a signer |
| `DOCUMENT_COMPLETED` | All signers have signed, document is complete |
| `DOCUMENT_FORCE_COMPLETED` | Document was manually force-completed |
| `DOCUMENT_CANCELLED` | Document was cancelled |
| `DOCUMENT_WITHDRAWN` | Document was withdrawn |
| `DOCUMENT_DELETED` | Document was deleted |
| `DOCUMENT_RESET` | Document was reset to draft |
| `DOCUMENT_RECREATED` | Document was recreated |
| `DOCUMENT_API_ACCESS` | Document was accessed via API |
| `DOCUMENT_UNAUTHORIZED_ACCESS` | Unauthorized access attempt |
