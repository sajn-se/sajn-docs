# Welcome to sajn

sajn is a modern digital document signing platform built for the Swedish market. Create, send, and manage legally binding digital signatures with ease.

## Key Features

<CardGroup cols={2}>
  <Card title="Digital Signatures" icon="pen-to-square">
    Send documents for signing via email or SMS with multiple signature methods including BankID integration.
  </Card>
  <Card title="Document Management" icon="folder-open">
    Organize documents with tags, custom fields, and powerful search capabilities.
  </Card>
  <Card title="Identity Verification" icon="shield-check">
    Verify identities using Swedish BankID with sajn ID - supporting both SSN-based and name-based verification.
  </Card>
  <Card title="API Integration" icon="code">
    Comprehensive REST API with full TypeScript support for seamless integration into your applications.
  </Card>
</CardGroup>

## Use Cases

### Contract Management
Send contracts for signing to multiple parties with sequential or parallel signing workflows.

### Employee Onboarding
Streamline your hiring process with digital document signing for employment contracts and agreements.

### Customer Agreements
Enable customers to sign agreements digitally, reducing paperwork and improving conversion rates.

### Identity Verification
Verify customer identities using Swedish BankID before granting access to services or completing transactions.
