## Det här kan du göra med sajn

<CardGroup cols={2}>
  <Card title="Digitala signaturer" icon="pen-to-square">
    Skicka dokument för signering via e-post eller SMS – med BankID, klicksignering eller ritad signatur.
  </Card>
  <Card title="Dokumenthantering" icon="folder-open">
    Organisera dokument med taggar, anpassade fält, mappar och kraftfull sökning.
  </Card>
  <Card title="Identifiering med BankID" icon="shield-check">
    Verifiera identiteter med svenskt BankID via SajnID – både personnummerbaserat och namnbaserat.
  </Card>
  <Card title="API & webhooks" icon="plug">
    Bygg integrationer mot sajn med vårt REST-API och realtidsuppdateringar via webhooks.
  </Card>
</CardGroup>
